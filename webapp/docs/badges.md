## Recommended tiering

| Tier     | Use for                       |
| -------- | ----------------------------- |
| Bronze   | First-time / easy badges      |
| Silver   | Repeated behaviour            |
| Gold     | Strong milestones             |
| Platinum | Rare / long-term achievements |

For example: `1 km Walked = Bronze`, `100 km Total = Gold`, `1,000 km Total = Platinum`.
